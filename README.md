# Şañıraq.kz — MVP маркетплейса жилых помещений

REST API для маркетплейса недвижимости Казахстана. Пользователи могут создавать объявления о жилье, оставлять комментарии и управлять своим профилем.

## Технологии

- **FastAPI** — веб-фреймворк
- **SQLAlchemy** — ORM для работы с базой данных
- **SQLite** — база данных
- **Alembic** — миграции
- **JWT (python-jose)** — аутентификация
- **bcrypt** — хеширование паролей
- **pytest** — тестирование
